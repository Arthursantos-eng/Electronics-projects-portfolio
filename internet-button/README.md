# Internet Button – Cloud Notification System

## Overview
A WiFi-connected button that sends a notification to a smartphone when pressed, used to inform a family member when leaving or arriving home.

## Hardware
- Particle Argon
- Push button
- Breadboard & wiring

## Architecture
Button → Argon → Particle Cloud → Smartphone notification

## Features
- Cloud API integration
- Event publishing with Particle Cloud
- Low-latency remote notification

## Skills Demonstrated
- RESTful IoT communication  
- Cloud-connected embedded systems  
- Event-driven firmware design
