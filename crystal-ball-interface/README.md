# Arduino LCD Interface

## Overview
A character LCD interface for displaying real-time system data from an Arduino microcontroller.

## Hardware
- Arduino Uno
- 16x2 LCD
- Potentiometer
- Breadboard wiring

## Features
- Dynamic text updates
- User-adjustable contrast control
- Parallel LCD communication

## Skills Demonstrated
- Peripheral interfacing  
- Timing-critical control signals  
- Embedded display drivers
